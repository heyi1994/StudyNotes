# 学习笔记

基于 [MkDocs](https://www.mkdocs.org/) + [Material 主题](https://squidfunk.github.io/mkdocs-material/) 构建的个人技术学习笔记，涵盖全栈开发、移动端、DevOps、逆向工程等领域。

## 内容

- **编程语言** — Rust、TypeScript、垃圾回收算法
- **前端** — React、Next.js、Redux、Zustand、Tailwind CSS、Sass
- **后端** — Docker
- **移动端** — Flutter、Android Compose、React Native
- **CI/CD** — GitHub Actions、GitLab CI/CD、Jenkins、Fastlane
- **逆向工程** — Frida、LSPosed、JADX
- **AI** — GitNexus

## 本地运行

```bash
pip install -r requirements.txt
mkdocs serve
```

浏览器访问 http://127.0.0.1:8000

## 构建

```bash
mkdocs build
```

静态文件输出到 `site/` 目录。
